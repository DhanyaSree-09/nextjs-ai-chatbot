# Next.js AI Chatbot

A simple AI-powered chatbot application built using Next.js 14 (App Router) and the AI SDK.
This project allows users to chat with an AI model through a clean and responsive interface.

# What This App Does

Users can type messages and get AI-generated replies

Chat history can be stored securely

Supports modern AI models through a unified API

Uses server-side rendering for better performance

# Features

Next.js 14 App Router for fast and modern routing

React Server Components & Server Actions for improved performance

AI SDK for generating AI responses

Clean UI using Tailwind CSS and shadcn/ui

Authentication using Auth.js

Chat history storage using a database

# Tech Stack

Framework: Next.js 14 (App Router)

Frontend: React, TypeScript, Tailwind CSS

AI Integration: Vercel AI SDK

Database: Neon Serverless PostgreSQL

Storage: Vercel Blob

Authentication: Auth.js

Deployment: Vercel

# Environment Variables

Environment variables are required for:
